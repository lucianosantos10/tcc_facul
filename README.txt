'Sistema para revisão de conteúdos com flashcards'
Projeto criado para o trabalho de conclusão de curso(tcc) superior. 
Curso de Análise e Desenvolvimento de Sistemas.   


Para rodar o sistema é preciso ter instalado 
as seguintes ferramentas: 

- NetBeans IDE 8.2 
- JDK 8u111 (Java SE Development Kit 8 Update 111)
- Servidor XAMPP
